# Contributing to CampusTrade

Thank you for your interest in contributing to **CampusTrade**! We welcome contributions from the student community to make this platform better for everyone.

1.  **Fork the Repository**: Create your own copy of the repository.
2.  **Clone Locally**: Download the code to your machine.
    ```bash
    git clone https://github.com/rah-gif/campus-trade.git
    ```
3.  **Create a Branch**: Always work on a new branch for your feature or bug fix.
    ```bash
    git checkout -b feature/amazing-feature
    ```

### Code Style

- We use **ESLint** and **Prettier** for code formatting.
- Ensure all new components are responsive (mobile-first approach).
- Use **Tailwind CSS** utility classes; avoid custom CSS unless necessary.

## Êèê‰∫§ (Submission) Guidelines

1.  **Commit Messages**: Write clear, descriptive commit messages.
    - Good: `feat(auth): add email domain validation`
    - Bad: `fix bug`
2.  **Pull Request (PR)**:
    - Push your changes to your fork.
    - Open a Pull Request against the `main` branch.
    - Provide a clear description of what you changed and why.
    - Attach screenshots if you modified the UI.
